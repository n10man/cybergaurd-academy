# Assets Directory

This directory contains all game assets for the CyberGuard Academy Phaser game.

## Structure

- `/maps` - Tilemap JSON files (Tiled map editor format)
- `/tilesets` - Tileset images used in maps
- `/sprites` - Character and object sprite sheets

## Usage

Assets in this directory are served from the `/assets` path and can be loaded by Phaser using relative paths from the public directory.
